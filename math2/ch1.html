1-8　文字式の利用（説明・証明）

STEP 2　基本

(1)
m、n を整数とすると、2つの偶数は 2m、2n と表せる。
和は 2m+2n = 2(m+n)。m+n は整数だから偶数。
よって、2つの偶数の和は偶数になる。
(2)
m、n を整数とすると、偶数は 2m、奇数は 2n+1 と表せる。
和は 2m+(2n+1) = 2(m+n)+1。m+n は整数だから奇数。
よって、偶数と奇数の和は奇数になる。
(3)
n を整数とすると、連続する2つの整数は n、n+1 と表せる。
和は n+(n+1) = 2n+1。n は整数だから奇数。
よって、連続する2つの整数の和は奇数になる。
(4)
n を整数とすると、連続する3つの整数は n、n+1、n+2 と表せる。
最大の数と最小の数の和は n+(n+2) = 2n+2 = 2(n+1)。
これはまん中の数 n+1 の2倍である。

STEP 3　標準

(1)
n を整数とすると、連続する2つの奇数は 2n+1、2n+3 と表せる。
和は (2n+1)+(2n+3) = 4n+4 = 4(n+1)。
n+1 は整数だから 4(n+1) は4の倍数。
よって、連続する2つの奇数の和は4の倍数になる。
(2)
十の位を a、一の位を b とすると、もとの数=10a+b、入れかえた数=10b+a。
差は (10a+b)−(10b+a) = 9a−9b = 9(a−b)。
a−b は整数だから 9(a−b) は9の倍数。
よって、その差は9の倍数になる。
(3)
m、n を整数とすると、3の倍数は 3m、6の倍数は 6n と表せる。
和は 3m+6n = 3(m+2n)。m+2n は整数だから3の倍数。
よって、3の倍数と6の倍数の和は3の倍数になる。
(4)
n を整数とすると、連続する3つの整数は n、n+1、n+2 と表せる。
(n+2)²−n² = (n²+4n+4)−n² = 4n+4 = 4(n+1)。
これはまん中の数 n+1 の4倍である。

STEP 4　応用

(1)
十の位を a、一の位を b とすると、自然数は 10a+b。
各位の和が3の倍数だから a+b=3k（k は整数）とおける。
10a+b = 9a+(a+b) = 9a+3k = 3(3a+k)。
3a+k は整数だから 3(3a+k) は3の倍数。
よって、その自然数は3の倍数になる。
(2)
n を整数とすると、連続する4つの整数は n、n+1、n+2、n+3 と表せる。
和は n+(n+1)+(n+2)+(n+3) = 4n+6 = 2(2n+3)。
2n+3 は整数だから 2(2n+3) は偶数。
よって、連続する4つの整数の和は偶数になる。
(3)
n を整数とすると、奇数は 2n+1 と表せる。
(2n+1)² = 4n²+4n+1 = 2(2n²+2n)+1。
2n²+2n は整数だから 2(2n²+2n)+1 は奇数。
よって、奇数の2乗は奇数になる。

STEP 5　確認テスト

(1)
n を整数とすると、連続する2つの整数は n、n+1 と表せる。
積に大きい方を加えると n(n+1)+(n+1) = (n+1)(n+1) = (n+1)²。
これは大きい方の数 n+1 の2乗である。
(2)
百の位を a、十の位を b、一の位を c とすると、自然数は 100a+10b+c。
各位の和が9の倍数だから a+b+c=9k（k は整数）とおける。
100a+10b+c = 99a+9b+(a+b+c) = 99a+9b+9k = 9(11a+b+k)。
11a+b+k は整数だから 9(11a+b+k) は9の倍数。
よって、その自然数は9の倍数になる。
「○を整数とすると」と最初に文字でおき、最後は 2×(整数) や 3×(整数) の形に変形して結論づける。式だけでなく、ことばの結論まで書くこと。
