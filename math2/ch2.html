2-4　連立方程式の利用（文章題）

STEP 2　基本

(1)
りんごを x 個、みかんを y 個買ったとすると、
　x + y = 13
　100x + 50y = 850
これを解いて x = 4、y = 9。
答え　りんご 4 個、みかん 9 個
(2)
りんごを x 個、みかんを y 個買ったとすると、
　x + y = 11
　80x + 60y = 800
これを解いて x = 7、y = 4。
答え　りんご 7 個、みかん 4 個
(3)
大人を x 人、子どもを y 人とすると、
　x + y = 15
　600x + 200y = 6200
これを解いて x = 8、y = 7。
答え　大人 8 人、子ども 7 人
(4)
ケーキを x 個、シュークリームを y 個買ったとすると、
　x + y = 12
　100x + 50y = 900
これを解いて x = 6、y = 6。
答え　ケーキ 6 個、シュークリーム 6 個

STEP 3　標準

(1)
十の位の数を x、一の位の数を y とすると、
　x + y = 3
　(10y + x) − (10x + y) = 9　→　9y − 9x = 9　→　y − x = 1
これを解いて x = 1、y = 2。
答え　もとの自然数は 12
(2)
十の位の数を x、一の位の数を y とすると、
　x + y = 10
　(10y + x) − (10x + y) = 18　→　9y − 9x = 18　→　y − x = 2
これを解いて x = 4、y = 6。
答え　もとの自然数は 46
(3)
行きにかかった時間を x 時間、帰りにかかった時間を y 時間とすると、
　x + y = 5
　4x = 6y　（行きと帰りの道のりは等しい）
4x=6y → 2x=3y。x=5−y を代入して 2(5−y)=3y → 10=5y → y=2、x=3。
答え　行き 3 時間、帰り 2 時間

STEP 4　応用

(1)
4%の食塩水を x g、9%の食塩水を y g とすると、
　x + y = 200
　0.04x + 0.09y = 0.06 × 200 = 12　（100倍して 4x + 9y = 1200）
これを解いて x = 120、y = 80。
答え　4%の食塩水 120 g、9%の食塩水 80 g
(2)
現在の父の年齢を x 歳、子の年齢を y 歳とすると、
　x + y = 46
　x + 10 = 2(y + 10)　→　x − 2y = 10
これを解いて x = 34、y = 12。
答え　父 34 歳、子 12 歳
(3)
昨年の男子を x 人、女子を y 人とすると、
　x + y = 500
　0.05x − 0.04y = 7　（100倍して 5x − 4y = 700）
これを解いて x = 300、y = 200。
答え　昨年の男子 300 人、女子 200 人

STEP 5　確認テスト

(1)
ケーキを x 個、シュークリームを y 個買ったとすると、
　x + y = 6
　120x + 50y = 440
これを解いて x = 2、y = 4。
答え　ケーキ 2 個、シュークリーム 4 個
(2)
十の位の数を x、一の位の数を y とすると、
　x + y = 10
　(10y + x) − (10x + y) = 54　→　9y − 9x = 54　→　y − x = 6
これを解いて x = 2、y = 8。
答え　もとの自然数は 28
単位（個・人・円・m・g・歳）をそろえて式を立てる。求めた x・y が問いの答えそのものとは限らない（『もとの数』などは作り直す）。
